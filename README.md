# what is this.

get perfect shadow every time for non-designer.

# Installation

`npm i shadow --save`

Then...

```
import { shadow } from 'shadow';

shadow({
    shadow_type: 'soft',
    padding: false
});

```

# options

shadow supports two optional, both of which are optional:

*shadow_type* - _hard | soft_  (Default to soft)
*padding* - _boolean_ (default to false)
